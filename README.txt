oStorybook

Which file to use?

Debian users : oStorybook-X.Y.Z.deb
RPM based users : ostorybook-X.Y.Z.rpm
All others Linux users : oStorybook-X.Y.Z-linux.bin
(install with : sudo sh -i oStorybook-X.Y.Z-linux.bin)
Mac OS X : oStorybook-X.Y.Z-macOSx.tgz
Windows users : oStorybook-X.Y.Z.exe
